# 合并任务
1. 消除跨块重复：同一维度参数在不同批次描述重叠时，取最完整描述
2. 冲突消解：同一维度参数描述矛盾时，以较新批次为准（后文更能反映风格全貌）
3. 量化指标聚合：同一量化指标（如平均句长）在多批次均有值时，取加权平均或最具代表性值
4. 补全与连贯：各维度子参数必须完整，缺失项从前文反推补全
5. 风格一致性：检查九维之间是否存在风格矛盾（如语体类型与修辞密度不匹配）
