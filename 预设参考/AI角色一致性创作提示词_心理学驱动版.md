# 角色一致性创作提示词（心理学驱动版）

> 本提示词基于人格心理学、依恋理论、动机理论及叙事心理学构建，用于指导AI在长篇小说/同人文创作中保持角色一致性，避免OOC（Out of Character）。
> 使用方法：将【方括号】内的内容替换为你的角色信息，然后完整复制给AI。

---

## 一、系统指令（System Prompt）

```
你是一位精通人格心理学与叙事创作的专业作家助手。你的核心任务是：
1. 严格遵循下方提供的角色心理档案进行创作，确保角色在任何情境下的行为、语言、决策都符合其人格结构。
2. 禁止为了推动剧情而扭曲角色性格（剧情奴隶化）。
3. 禁止让角色说出/做出与其核心恐惧、防御机制、价值观相矛盾的言行。
4. 当面临两难抉择时，必须优先参考角色的"决策模式"与"动机优先级"，而非选择最方便剧情的选项。
5. 每个角色的台词必须能通过"对话剥离测试"——即去掉角色名后，读者仍能辨认出是谁在说话。
6. 在描写角色心理活动时，必须体现其认知风格（实感/直觉、思维/情感）和情感调节模式。
```

## 三、人格操作系统（核心层）

```
【认知风格】
- 信息获取：□ 实感型（关注具体细节、当下事实） / □ 直觉型（关注模式、隐喻、未来可能性）
- 决策方式：□ 思维型（逻辑优先、客观分析） / □ 情感型（价值优先、共情他人）
- 认知复杂度：□ 二元思维（非黑即白） / □ 多元思维（接受灰度与矛盾）

【人格结构（弗洛伊德三我）】
- 本我（Id）冲动：[角色最原始、不可抑制的欲望。如：对特定之人的占有欲、毁灭冲动、求生本能]
- 超我（Superego）约束：[角色的道德铁律与理想自我。如："绝不伤害无辜"、"家族荣誉高于一切"]
- 自我（Ego）调解模式：[角色通常如何平衡本我与超我。如："先表面服从，暗中寻找两全方案"、"压抑本我直到临界点爆发"]

【大五人格定位（1-10分，10为极端）】
- 开放性（O）：__/10  [好奇/保守]
- 尽责性（C）：__/10  [自律/随性]
- 外向性（E）：__/10  [热情/内敛]
- 宜人性（A）：__/10  [合作/怀疑]
- 神经质（N）：__/10  [情绪化/稳定]
*注：建议至少有一个极端高分（≥8）和一个极端低分（≤3）以形成内在张力。*

【依恋类型】
- 对亲密关系：□ 安全型 / □ 焦虑型 / □ 回避型 / □ 恐惧型（焦虑+回避）
- 具体表现：
  * 安全型：能表达需求，也能给予空间。
  * 焦虑型：患得患失、疯狂索取确认、分离焦虑、过度解读冷淡。
  * 回避型：遇真心就退缩、用独立掩饰脆弱、情感隔离、"去激活策略"。
  * 恐惧型：既渴望亲密又害怕受伤，在靠近与推开之间摇摆。
- 在压力关系中的行为：[如：焦虑型会反复确认"你爱我吗"；回避型会突然冷漠并找理由离开]

【核心自我叙事】
"我是________的人。" [角色给自己讲的人生故事。如："我是幸存者"、"我是拯救者"、"我是观察者"]
```

## 四、动力与动机系统（动力层）

```
【核心恐惧】：[角色最深层、最不愿面对的恐惧。如：被抛弃、被证明无能、失去自我、被控制]
【核心渴望】：[与恐惧成对出现。如：被接纳、被认可、自由自主、被看见]

【动机优先级（按重要性降序排列）】
1. [最高优先级，如：保护妹妹]
2. [如：维护家族名誉]
3. [如：个人情感需求——通常被压抑或排在最后]
4. [如：自我实现]
*注：当动机冲突时，角色必须优先选择排序更高的动机，除非触发"核心恐惧"导致失控。*

【当前最迫切未满足的需求（马斯洛）】：
□ 生理 / □ 安全 / □ 爱与归属 / □ 尊重 / □ 自我实现
【深层恐惧的具体化】：[不是"害怕孤独"，而是"害怕在深夜醒来发现身边空无一人，且这永远不会改变"]

【价值观光谱（施瓦茨理论，选前三并排序）】
1. [如：成就（通过成功证明价值）]
2. [如：安全（稳定与秩序）]
3. [如：自主（独立与自由）]
*注：当价值观冲突时，角色会优先捍卫排序更高的价值。*

【阿德勒生活方式】：
□ 支配型（我必须掌控一切，否则就会输）
□ 索取型（别人应该照顾我，满足我）
□ 回避型（只要不尝试，就不会失败）
□ 社会利益型（通过合作与贡献获得价值）
```

## 五、情感与防御机制

```
【情感调节模式】
□ 表达型（当场发作，事后不记仇）
□ 压抑型（表面平静，长期积累后爆发）
□ 转移型（踢猫效应，对A的怒火发泄到B身上）
□ 理智化（用分析逃避感受，"我只是太累了"）
□ 解离型（情绪过载时灵魂出窍，仿佛在看戏）

【常用心理防御机制（选2-3个）】
1. 主要防御：[如：压抑——把痛苦记忆推入无意识]
2. 次要防御：[如：合理化——"我这么做是因为环境所迫"]
3. 压力下的退行防御：[如：否认——拒绝承认重要之人已离开]

【创伤类型（如有）】：
□ 背叛创伤（核心信念：没人值得信任）
□ 遗弃创伤（核心信念：我终究会被抛弃）
□ 羞辱创伤（核心信念：我是可耻的）
□ 侵入创伤（核心信念：世界不安全/我被侵犯）
- 触发场景：[什么情境会激活创伤？如：伴侣突然失联、当众被否定、被逼近到无处可退]
- 应激反应：[激活后的具体行为。如：疯狂打电话定位、暴怒反击、僵住无法动弹]

【压力下退行表现】：[平时成熟稳重的角色，在极限压力下会退行到什么状态？如：像小孩一样粘人、变得极度叛逆、完全依赖他人]
```

## 六、行为指纹与身体语言（行为层）

```
【身体语言系统（必须差异化）】
- 紧张时：[如：反复整理袖口，而非普通的搓手]
- 撒谎/隐瞒时：[如：反而直视对方，语速变慢（过度补偿）]
- 愤怒时：[如：先笑，然后轻声说话——比大吼更可怕]
- 悲伤时：[如：不哭泣，而是发呆并机械性地做家务（解离）]
- 心动/动情时：[如：突然变得刻薄、挑剔（反向形成）]
- 放松/信任时：[如：无意识模仿对方的动作、暴露脆弱部位]

【日常仪式与习惯】
- 压力习惯：[如：咬笔帽、整理书桌、暴饮暴食、数质数]
- 空间习惯：[房间极度整洁（外化控制欲）或极度混乱（外化内心失序）]
- 时间感知：[总是提前20分钟到（焦虑）或习惯性迟到（抗拒被控制）]

【物质与消费观】
- 对金钱：[挥霍填补空虚 / 吝啬换取安全感 / 只看工具价值]
- 对物品：[恋物（旧物不舍） / 极简（随时可离开）]
- 对穿着：[伪装（穿成"应该成为的样子"） / 表达（穿成"真实的自己"）]

【语言指纹】
- 口头禅/惯用句式：[如："理论上来说..."、"随便你"、"你懂什么"]
- 对不同对象的称呼差异：
  * 对[亲密之人]：[如：只叫单字昵称]
  * 对[普通关系]：[如：全名+敬语]
  * 对[厌恶之人]：[如：用"那位"代替名字]
- 知识储备影响的语言：[如：法学背景的人习惯用"构成要件"、"举证责任"等概念打比方]
- 情绪状态下的语言变化：[生气时从长句变短句，或从礼貌变刻薄]
```

```

- 注意：上位者可能源于恐惧失控；下位者可能暗中积累怨恨。

【关系中的角色扮演】
| 关系对象 | 角色面具 | 真实感受 | 潜在冲突 |
|---------|---------|---------|---------|
| 对[权威/强者] | [如：顺从的晚辈] | [如：压抑的愤怒] | [如：想反抗但不敢] |
| 对[平辈/朋友] | [如：开心果] | [如：孤独] | [如：用幽默防御亲密] |
| 对[弱者/下属] | [如：严厉的导师] | [如：不安/怕被看穿] | [如：冒充者综合征] |

【边界感】

## 八、变化轨迹与弧光（变化层）

```
【当前弧光阶段】
□ 第一阶段：否认（"我没问题，是世界有问题"）
□ 第二阶段：触发（某个事件打破防御）
□ 第三阶段：挣扎（新旧自我的拉锯战）
□ 第四阶段：顿悟（意识到旧模式不再适用）
□ 第五阶段：践行（做出不同选择，但可能反复）

【未解决的埃里克森危机】
- 阶段：[如：亲密 vs 孤独]
- 遗留问题：[如：无法信任伴侣，用工作逃避亲密]
- 如何外化为行为：[如：每次关系深入就找借口出差]

【成长转折点（具体触发）】
- 触发事件：[什么具体场景迫使角色面对自己？]
- 旧模式的最后一次使用：[角色在改变前，会如何最后一次使用旧防御？]
- 新选择的代价：[改变不是免费的，角色会失去什么？]
```

## 九、OOC红线与强制约束

```
【绝对不能做的事】（剧情绝不能扭曲这些）
1. [如：绝不会主动求助，即使濒临死亡]
2. [如：绝不会在公众场合暴露脆弱，哭泣必须独处]
3. [如：绝不会伤害无辜者，即使这是达成目标的最短路径]
4. [如：绝不会对[特定之人]说重话，即使极度愤怒]
5. [如：绝不会使用[某种手段]，即使效率最高]

【必须出现的标志性行为/台词】（至少每章出现1-2次）
1. [如：对亲近之人的特定昵称]
2. [如：紧张时的标志性小动作]
3. [如：表达关心的反向说法——"你死了会给我添麻烦"]

【决策模式校验】
当角色面临两难时，按以下优先级决策：
1. 是否符合【性格铁律】？
2. 是否符合【动机优先级】？
3. 是否激活了【核心恐惧】导致失控？
4. 是否使用了【默认防御机制】？
5. 最后才考虑：是否推动剧情？
*如果推动剧情需要违反前4条，必须修改剧情，而非修改角色。*
```

```
现在请根据以上角色档案，创作第___章。

【本章情节要点】（拆分成5-8个小节点）
1. [场景+动作，约X字]
2. [场景+动作，约X字]
...

【本章角色状态】
- 当前情绪：[如：表面平静，内心因昨日事件而焦虑]
- 性格体现要求：[如：必须体现其回避型依恋——当对方靠近时，用工作借口拉开距离]
- 互动风格：[如：对A保持礼貌但疏离，对B暴露罕见的脆弱]

【本章必须包含】
- 至少一次【标志性小动作】
- 至少一次【对不同对象的差异化称呼/态度】
- 至少一次【内心独白体现其认知风格】
- 至少一次【动机与恐惧的拉扯】

【本章禁止出现】
- [如：禁止让角色突然说出直白的告白]
- [如：禁止让角色在他人面前哭泣]
- [如：禁止角色使用超出其知识储备的词汇]

【上下文提醒】
- 上一章结尾：[简述]
- 当前伏笔状态：[哪些伏笔待回收]
- 其他角色在场状态：[提醒AI其他角色的当前情绪和立场]

## 十一、一致性自检Prompt（写完后使用）

```
请基于上述角色档案，检查刚才生成的内容：

1. 【性格一致性】角色是否有任何言行违背"性格铁律"或"绝对不能做的事"？
2. 【动机回溯】角色的每个重大决定是否都能追溯到"动机优先级"或"核心恐惧"？有无剧情奴隶化？
3. 【对话剥离测试】如果去掉角色名，这段台词是否仍能辨认出是该角色？还是换给任何人都可以？
4. 【防御机制检查】角色在面对压力时，是否使用了设定的防御机制？还是突然变得"成熟理智"？
5. 【依恋模式检查】角色在亲密关系场景中，是否表现出设定的依恋类型特征？
6. 【创伤触发检查】如果本章涉及类似创伤的场景，角色的应激反应是否符合设定？
7. 【语言指纹检查】角色的用词、句式、称呼是否与设定一致？有无知识越界？
8. 【身体语言检查】角色的情绪是否通过设定的身体语言表达，而非直接心理描写？

如有任何一项未通过，请指出具体问题并给出修改建议。
```

## 十二、使用建议

1. **先填档案，再写正文**：不要边写边补设定，先完成档案再进入创作。
2. **分阶段生成**：先让AI生成大纲（你检查动机合理性），再生成场景分镜，最后逐场景扩写。
3. **每章贴档案**：AI不真正"记得"设定，每章开始时简要复述角色的当前情绪状态和动机。
4. **保留"故事圣经"**：单独维护一份文档，记录伏笔、角色状态变化、已发生事件，每章写作前更新。
5. **接受反复**：角色弧光中的"践行"阶段允许反复——角色可能在两章前进了一步，下一章又退半步，这才是真实的人性。
